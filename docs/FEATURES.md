# 🧑‍💻 HoneyRest – 주요 기능

---

## 👤 사용자 (User)

| 기능 | 설명 |
|------|------|
| 회원가입 / 로그인 | 이메일 인증 기반 가입, JWT 토큰 인증 |
| 소셜 로그인 | Google, Kakao OAuth2 |
| 숙소 검색 | 지역 / 카테고리 / 날짜 / 인원 필터 검색 |
| 인기 숙소 | Redis ZSet 기반 카테고리별 인기 숙소 랭킹 |
| 숙소 상세 | 객실 목록, 이미지, 편의시설, 위치 정보 |
| 예약 | 날짜 선택 → 객실 확인 → 결제 → 예약 완료 |
| 비회원 예약 | 예약번호 + 전화번호 기반 비회원 예약 조회 |
| Toss 결제 | 카드 / 가상계좌 결제, 영수증 URL 저장 |
| 예약 취소 | 취소 요청 및 상태 관리 |
| 리뷰 | 예약 완료 후 별점 + 내용 작성, 좋아요, 삭제 |
| 관심 숙소 | 위시리스트 저장 / 해제 |
| 마이페이지 | 예약 내역, 리뷰 목록, 프로필 수정, 쿠폰 확인 |
| 포인트 | 결제 시 포인트 차감 |
| 이메일 알림 | 예약 완료 시 비동기 이메일 발송 |

## 🖥️ 주요 화면

### 🏠 메인 페이지

![메인 화면](https://github.com/user-attachments/assets/a5e85b19-a87c-47b0-8b9b-91e9d3db2dd1)

숙소 검색, 지역 필터, 인기 숙소, 배너 등 핵심 기능이 배치된 메인 화면.

### 📅 예약 흐름

> 숙소 선택 → 객실 확인 → 날짜 선택 → 결제 → 예약 완료

- Toss Payments 연동으로 간편 결제
- 예약 완료 후 이메일 알림 및 마이페이지에서 예약 내역 확인 가능

### 🧑‍💼 마이페이지

- 예약 내역, 리뷰 작성, 프로필 수정, 쿠폰 확인 등
- 사용자 경험을 고려한 기능 배치
